AlmaLinux


¿Que es AlmaLinux?

AlmaLinux es un nueva distribución del sistema operativo Linux, basada también en código abierto, que deriva de CentOS; y será una de sus reemplazantes a partir de la desición de Red Hat (su desarrollador) de dejar de ofrecer soporte sobre CentOS desde el 31 de diciembre de 2021, donde pasará a ser una versión “steam” o de desarrollo para la distribución corporativa Red Hat Enterprise Linux (RHEL).

Desarrollo de AlmaLinux

Alma Linux se desarrolló en estrecha colaboración con la comunidad de Linux para llenar el vacío que ha dejado CentOS, dijo Jack About boul. Después de la versión beta del código, el sistema operativo AlmaLinux se sometió a pruebas exhaustivas para cumplir con las expectativas del usuario en cuanto a confiabilidad y rendimiento. La mayor parte del trabajo implicó depurar, agregar paquetes y preparar software para cargas de trabajo en producción. Además de las correcciones de errores conocidos, la versión estable incluye soporte para Errata, SecureBoot, AWS y VMware, así como scripts para una migración rápida de CentOS a AlmaLinux.

¿Para que esta diseñado AlmaLinux?

AlmaLinux da la imagen de ser un sistema operativo empresarial y, por lo tanto, está dirigido en particular a las empresas y organizaciones que buscan una distribución de Linux estable y bien mantenida para fines comerciales. AlmaLinux aparenta ser explícitamente una alternativa gratuita a Red Hat Enterprise Linux, que es de pago. Los usuarios pueden utilizar AlmaLinux para construir una arquitectura operativa segura y optimizada para las aplicaciones. En particular, los servidores que necesitan un sistema operativo fiable tienen una muy buena solución con esta distribución Linux. El sistema se utiliza en entornos físicos, virtuales y en la nube. A pesar de su enfoque empresarial, AlmaLinux también es adecuado para uso privado.
